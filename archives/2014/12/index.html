Vue.js

Vue.js 是一个用于创建 web 交互界面的库。
它具有简单而灵活的 API，创建出来的组件可以实时反映数据的变化。

安装 Vue.js v0.12.8 源代码在 GitHub 上

  • Follow @vuejs

简单

写点 HTML，拿来 JSON，创建一个 Vue 示例，就这么简单。

可反应

表达式 & 无需声明依赖的可推导属性 (computed properties)。

组件化

用解耦的、可复用的组件组合你的应用程序。

紧凑

~22kb min+gzip，且无依赖。

快速

以精确有效的异步批处理方式更新 DOM。

已打包

可以通过 NPM、Bower 或 Duo 安装——无缝融入你的生命！
